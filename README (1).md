# Football Ticket Booking System

## 📌 Project Overview

This project is a simple Football Ticket Booking System developed using PostgreSQL. It demonstrates database design, ERD creation, table relationships, and SQL queries.

---

## 📂 Database Tables

### Users

Stores information about football fans and ticket managers.

### Matches

Stores football match information.

### Bookings

Stores ticket booking records.

---

## 🔗 Relationships

- One User → Many Bookings
- One Match → Many Bookings
